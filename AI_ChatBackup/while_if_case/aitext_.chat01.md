Python

Python
# for loop
for i in range(5):
  print(f"Iteration: {i}")

# while loop
count = 0
while count < 3:
  print(f"Counting: {count}")
  count += 1

# if statement
age = 20
if age >= 18:
  print("You are an adult.")
else:
  print("You are a minor.")
Use code with caution.
